VIRTUAL MOUSE PROGRAM

This program aims to use Mouse using hand gestures. 

How to use :

> Install all the requirement files using

  pip3 install -r requirements.txt

> Run the program (this will start a screen with a box that will be a screen representation for the   mouse in which it can be controlled by hand gesture)
> To use the hand gesture use Right hand 
  USE the INDEX and MIDDLE FINGER WHILE MAKING A "V" TO NAVIGATE and to make a action bring both   fingers   close to click the key(like make a scissor gesture in vertical position)
> once done you can hard close the program by pressing ctrl+c on the terminal 
> This may not run in full screen 
> To control its movement smoothly try changing the smoothing value
