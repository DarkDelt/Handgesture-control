IMAGE SLIDE PROGRAM

This program aims to use images using hand gestures. 

How to use :

> Install all the requirement files using

  pip3 install -r requirements.txt

> Run the program (this will start a screen with images in the folder these images can be changed)
> To use the hand gesture use Right hand 
  USE the thumb to move the image to left side for going to previous image
  USE the little finger to move to the right for forwarding the image
> once done you can hard close the program by pressing ctrl+c on the terminal or use "`" key to close
